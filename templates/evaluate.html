The probes ask factual questions about the same two distributions the model is shown above — which category is largest, which gained most, did X rise or fall — with answers computable directly from the display. None pass through the theme taxonomy, the five-point threshold, or the single-label formulation. The model reads these distributions almost perfectly; what it does not reproduce is the label the rule assigns to them. The label accuracy is therefore a measurement of the schema rather than of the model. Probes are clustered ({{ probe.n_cities }} cities × ~{{ (probe.n_items / probe.n_cities)|round|int }} each), so they are not {{ probe.n_items }} independent observations.
